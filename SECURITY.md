# Security Policy

## Reporting a vulnerability

Please report security issues privately via
[GitHub Security Advisories](https://github.com/deebee-tech/sqleasy/security/advisories/new)
for this repository.

If advisories are unavailable, open a
[private security report / issue](https://github.com/deebee-tech/sqleasy/issues/new)
and mark it as a security concern — do not post exploit details in a public issue.

We will acknowledge reports and work on a fix before any public disclosure.
